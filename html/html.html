  • HTML-Boilerplate: This is the basic structure of HTML 
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Website</title>
</head>
<body>
                    
</body>
</html>
  • Headings: There are six headings available in HTML.Among which h1 is the largest and h6 is the smallest 
    <h1> Tag
<h1> Heading 1 </h1>
    <h2> Tag
<h2> Heading 2 </h2>
    <h3> Tag
<h3> Heading 3 </h3>
    <h4> Tag
<h4> Heading 4 </h4>
    <h5> Tag
<h5> Heading 5 </h5>
    <h6> Tag
<h6> Heading 6 </h6>
  • Container: Container tags are the tags that contain some data such as text, image, etc. There are several container tags in HTML 
    <div> This is div block </div>
    <span> This is span block </span>
    <p> This is a paragraph </p>
    <pre> This is pre block </pre>
    <code> This is code block </code>
  • Text-Formatting: Text formatting tags are used to format text or data of HTML documents. You can do certain things like creating italic, bold, strong text to make your document look more attractive and understandable 
    <b> I'm Bold text </b>
    <strong> I'm Strong text </strong>
    <i> I'm Italic text </i>
    <em> I'm Emphasized text </em>
    <sub> I'm Subscript text </sub>
    <sup> I'm Superscript text </sup>
